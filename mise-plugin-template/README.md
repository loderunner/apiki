# asdf-apiki

[asdf](https://asdf-vm.com/) / [mise](https://mise.jdx.dev/) plugin for [apiki](https://github.com/loderunner/apiki)

## Install

```bash
asdf plugin add apiki https://github.com/loderunner/asdf-apiki.git
# or with mise:
mise use -g apiki
```

## Use

```bash
asdf install apiki latest
asdf global apiki latest
# or with mise:
mise install apiki
```

## Requirements

- curl
- tar
- bash
